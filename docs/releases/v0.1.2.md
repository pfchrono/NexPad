NexPad v0.1.2
=================

Draft release notes for the next patch cycle.

What Changed
======

* Added internal padding to the app's multiline help and log edit controls, plus the single-line edit boxes, to improve readability and make the UI feel less cramped.
* Added SHA256 checksum generation for packaged Win32 and x64 release zip artifacts.
* Added checksum files to the GitHub Actions artifact upload and GitHub Release publishing flow.
* Added CI verification so generated checksum files are checked against the built zip files before upload.
* Updated release documentation and the release checklist to include checksum verification.
* Updated the checked-in `v0.1.0` release notes to match the newer release-note format used by later releases.
* Added `docs/releases/TEMPLATE.md` as the reusable starting point for future release notes, including checksum verification guidance.

Validation
======

* Verified the UI padding patch builds cleanly in `Debug|Win32`.
* Verified local release packaging produces both zip archives and matching `.sha256` files.
* Verified the checksum verification script passes against the generated artifact set.

Expected Included Assets
======

* `NexPad-win32-v0.1.2.zip`
* `NexPad-win32-v0.1.2.zip.sha256`
* `NexPad-x64-v0.1.2.zip`
* `NexPad-x64-v0.1.2.zip.sha256`

Checksum Verification
======

Windows PowerShell example:

```powershell
$expected = (Get-Content .\NexPad-win32-v0.1.2.zip.sha256).Split(' ')[0]
$actual = (Get-FileHash .\NexPad-win32-v0.1.2.zip -Algorithm SHA256).Hash.ToLower()
$actual -eq $expected
```

Repeat the same check for the x64 archive before distribution or use.

Notes
======

This release is intended to improve release integrity and make packaged downloads easier to validate.
