NexPad v0.1.6
=================

UI themes and touchpad scroll tuning.

What Changed
======

* Added light and high-contrast UI theme options selectable from the Settings tab
  and via the new UI_THEME_MODE config key (0 = dark, 1 = light, 2 = high contrast).
* Added independent touchpad scroll controls: TOUCHPAD_SCROLL_ENABLED and
  TOUCHPAD_SCROLL_SPEED let you tune or disable DualSense two-finger scroll
  behavior without affecting one-finger cursor movement.

Validation
======

* Verified full solution matrix builds with scripts/build-all.ps1.
* Verified local release packaging with scripts/package-release.ps1 -Version v0.1.6.
* Verified checksum validation with scripts/verify-release-checksums.ps1.

Included Assets
======

* NexPad-win32-v0.1.6.zip
* NexPad-win32-v0.1.6.zip.sha256
* NexPad-x64-v0.1.6.zip
* NexPad-x64-v0.1.6.zip.sha256

Checksum Verification
======

Windows PowerShell example:

```powershell
$expected = (Get-Content .\NexPad-win32-v0.1.6.zip.sha256).Split(' ')[0]
$actual = (Get-FileHash .\NexPad-win32-v0.1.6.zip -Algorithm SHA256).Hash.ToLower()
$actual -eq $expected
```

Repeat the same check for the x64 archive before distribution or use.

Notes
======

This release adds UI theme flexibility for different lighting environments and
display preferences, and gives DualSense touchpad users independent control over
scroll sensitivity without affecting pointer speed.
