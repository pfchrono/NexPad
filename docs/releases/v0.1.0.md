NexPad v0.1.0
=================

First public NexPad release.

Highlights
======

* Rebranded the project from the earlier Gopher360 codebase into the standalone NexPad repository
* Renamed the Windows solution, project, binary, icon, and core application class to NexPad
* Added a native Windows configuration UI for status, settings, mappings, and preset management
* Added preset import, export, save, delete, and live configuration support
* Added GitHub Actions build automation for `main` and version tags
* Added automated release packaging for Win32 and x64 zip artifacts

Included Assets
======

* `NexPad-win32-v0.1.0.zip`
* `NexPad-x64-v0.1.0.zip`

Notes
======

This release establishes the standalone NexPad repository, packaging flow, and GitHub release automation. It is the baseline version for future iteration under the NexPad name.

Known Follow-up Areas
======

* Add checksums for packaged assets
* Improve release-note generation and documentation assets
* Continue smoothing first-run and setup guidance for end users
